Description

This experiment demonstrates a basic Traffic Light Controller using Arduino UNO. Three LEDs are used to represent the Red, Yellow, and Green traffic signals. The LEDs are activated in a predefined sequence to simulate the operation of a real-world traffic light system.

Components Used

- Arduino UNO
- Red LED
- Yellow LED
- Green LED
- Resistors
- Breadboard
- Jumper Wires

Working Principle

1. The Red LED turns ON to indicate STOP.
2. The Yellow LED turns ON to indicate WAIT.
3. The Green LED turns ON to indicate GO.
4. The sequence repeats continuously to simulate traffic signal operation.

Learning Outcome

- Understanding digital output control
- Controlling multiple LEDs using Arduino
- Implementing sequential logic
- Simulating a basic traffic signal system
