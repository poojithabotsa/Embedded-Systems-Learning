Description

This experiment demonstrates PWM (Pulse Width Modulation) using Arduino UNO to control the brightness of an LED.

By varying the duty cycle of the PWM signal, the LED brightness can be adjusted smoothly from dim to bright.

Components Used

- Arduino UNO
- LED
- Resistor
- Breadboard
- Jumper Wires

Learning Outcome

- Understanding PWM (Pulse Width Modulation)
- Controlling LED brightness using software
- Using Arduino PWM pins
- Understanding duty cycle and its effect on output
